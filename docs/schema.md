# `deck.json` schema

The deck file lives at `~/Library/Application Support/Bento/deck.json`. Bento hot-reloads it within a second of save, so you can edit it in any editor.

## Envelope

```json
{
  "schemaVersion": 1,
  "tiles": [ /* array of Tile objects */ ]
}
```

The `schemaVersion` is mandatory. Future versions of Bento will use it to migrate older decks forward — your hand-edited file should always declare a version.

## Tile

```json
{
  "id": "5C2C2A50-8B5F-4E89-9D62-7B3F3A2B1B3F",
  "label": "Coffee",
  "symbol": "cup.and.saucer.fill",
  "tint": "accent",
  "action": {
    "kind": "shell",
    "payload": "caffeinate -d -t 3600 &"
  },
  "liveKind": "caffeinate"
}
```

| Field      | Type    | Required | Notes |
|------------|---------|----------|-------|
| `id`       | UUID string | Yes (generated by the editor) | Stable identifier — you can press tiles by ID over the URL scheme or CLI |
| `label`    | String | Yes | One word reads best. Used by `bento press <label>` (slugified). |
| `symbol`   | String | Yes | Any [SF Symbol](https://developer.apple.com/sf-symbols/) name. Renders as a 22pt icon. |
| `tint`     | String | Yes | `neutral` (default), `accent`, or `red`. |
| `action`   | Object | Yes | See below. |
| `liveKind` | String | No | `caffeinate`, `focus`, or `mic`. Opts the tile into the live-state pattern. |

## Action

```json
{ "kind": "shell", "payload": "echo hello" }
{ "kind": "app", "payload": "/Applications/Calculator.app" }
{ "kind": "url", "payload": "https://example.com" }
```

### `shell`
Runs `/bin/zsh -ilc <payload>`. The `-i` (interactive) + `-l` (login) flags source both `~/.zprofile` and `~/.zshrc`, so Homebrew/nvm/pyenv tools work the way they do in your terminal.

### `app`
`open` the .app at the given path.

### `url`
`open` the given URL. Works for `https://`, `mailto:`, custom schemes (`raycast://`, `vscode://`, etc.).

## Validation

The app falls back to the default deck if `deck.json` fails to parse, and renames the broken file to `deck.corrupted.<timestamp>.json` so you don't lose your edits.

`bento doctor` prints the path and current tile count for sanity checks.
